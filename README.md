# Exploratory Binary & Multiclass Classification on Sleep Health and Lifestyle Data

This repo compares classification approaches using the following models:
kNN, Decision Tree, Random Forest, Naive Bayes, ANN, SVM

### Binary Problem: 
response: Stress Level
- Stress Level <= 5: Low 
- Stress Level > 5: High

### Multi Class Problem:
response: Quality of Sleep
- Labels 4-9 (higher is better) as proxy for sleep quality

## Dataset: Sleep & Lifestyle Study
https://www.kaggle.com/datasets/ayeshaimran1619/sleep-and-lifestyle-study/data
